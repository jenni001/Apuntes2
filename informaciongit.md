 # Información git
 
 En este *documento MD* redactaré un listado de comandos que deben utilizarse por consola en **comandos git**:

 ## Comandos git:

1. git init
2. git status
3. git add (. / un fichero en particular / *.txt)
4. git commit -m "comentario"
5. git log -oneline 
6. git resert --hard 
7. git push
